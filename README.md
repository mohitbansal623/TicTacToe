# Tic-Tac-Toe Game

Tic-tac-toe (also known as noughts and crosses or Xs and Os) is a game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

### About this game :
  - This is a single player game in which computer will defend player's move.
  - Both can win depending upon the moves they play.
 
#### Working :
> By click on start button, the game starts. The chance by default will be of player.
> After setting the move by the player, the comp player will check all possible cases 
> and resist player to win. 
> 
>
> In .html file, content is defined which is to be displayed on web page.
> In .css file, style is defined for the components designed in .html file. 
> In .js file, whole working of the game takes place.
> Functions and flag_variables/counters have been used to handle all cases.
